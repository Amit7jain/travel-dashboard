# UI/UX Coding Standards
- **Style:** Apple-esque (High white space, #FFFFFF backgrounds, Indigo-600 accents).
- **Icons:** Always use `lucide-react`.
- **Animations:** Use `framer-motion` for all transitions (use 'spring' for modals).
- **Responsiveness:** Mobile-first. Use `lg:` prefix for desktop-specific layouts.
- **Components:** Functional React components with TypeScript.
